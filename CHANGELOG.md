# Changelog

## 0.1.0 - Initial public release

- Created cleaned research-software repository structure.
- Added public Zenodo dataset contract and legacy filename mapping.
- Added CAE architecture and model-loading helpers.
- Added feature-map extraction, reference-map generation and NCC scoring.
- Added classifier training/evaluation utilities.
- Added scripts for dataset download/checking, CAE training, evaluation and benchmarking.
- Added pre-trained model asset installation helper.
- Added advanced raw-audio-to-WST workflow.
- Added prediction script for new audio/features.
- Added documentation for dataset, method, reproducibility, the included pre-trained model and new-data workflows.
- Added GitHub Actions test workflow, issue templates and pull-request template.
- Added unit tests using small synthetic arrays.
